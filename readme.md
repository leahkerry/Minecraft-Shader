# Our minecraft shader!!
Authors: Leah, Lydia, and Halle :)

## How to use

1. Download the [Iris shaderpack universal jar](https://www.irisshaders.dev/download) 
3. Run the installation for the Iris download
1. Download the [.zip](https://github.com/leahkerry/Minecraft-Shader/archive/refs/heads/main.zip) file of this repo
2. In the minecraft launcher, open the `Installations` tab. Click on the folder icon next to the iris shaderpack to open the folder that stores Iris
3. Go into the `shaderpacks` folder and extract the .zip file of this repo there  
4. Launch minecraft and open a world!
5. In minecraft, go to Options → Video Settings → Shader Packs → Open shader pack folder... Pick the location of this shader that you just downloaded. Make sure the Shaders: Enabled option is set
8. Play!! Adjust settings if needed

### Adjusting settings
Options → Video settings → Shader packs → (Make sure this shader is selected) → Shader pack settings.
